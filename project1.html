Recipe Site

In code bootcamp we were able to make an individual project outside of our normal class assignments. I chose to make a recipe website. The first version used HTML, Sass, jQuery, and Bootstrap to practice these technologies. Later I rebuilt the site using React using an API to display recipe data. The Homepage illustration was made in procreate.
